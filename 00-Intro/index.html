Дискрипторы логического форматирования

Дескрипторы логического форматирования нужны для анализа страницы поисковыми машинами. Многие из них замищают дескрипторы физического форматирования

<strong>(Bold) ― этот десткриптор делает текст жирным
<em>(Emphasis) ― этот десткриптор делает текст курсивом
<del>(Strikeout) ― этот десткриптор делает текст перечеркнутым
<code>(Code) ― этот десткриптор делает текст моноширным
<mark>(Mark) ― этот десткриптор делает текст моноширным
<cite>(Citation) ― этот десткриптор обозначает цитаты
